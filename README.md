# Aplira

AI-powered OPT job tracker Chrome extension.

## Overview
Aplira helps F1/OPT students:
- Track job applications
- Monitor OPT unemployment days
- Tag visa-friendly roles
- Generate AI-tailored resume bullets

## Tech Stack
- Chrome Extension (Manifest V3)
- React + TypeScript
- Tailwind CSS
- Supabase (Auth + Postgres)
- Serverless AI proxy
